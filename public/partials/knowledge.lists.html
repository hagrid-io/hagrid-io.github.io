  • List types

  • List: .list-{types}e.g: .input.btn-primary, .input.btn-error 
    <!-- primary input -->
<div class="input list-primary">

</div>
Use .input.list-{types} classes to modify the apparence of a input
  • List Sizes
  • Type: class
  • Dependencies: .input
  • Examples
  • List: .list-{sizes}e.g: .input.list-sm, .input.list-lg
    <!-- a small input -->
<div class="input list-sm">
  <input type="text" placeholder="Small input">
</div>
<!-- a info type and large input -->
<div class="input list-info list-lg">
  <input type="text" placeholder="Info type and large input">
</div>
You can use .input.list-{sizes} to change size input, available Sizes {xs, sm, lg, xl}
  • List Looks
  • Type: class
  • Dependencies: .input
  • Examples
  • List: .list-{looks}e.g: .input.list-upper, .input.list-outline
    <!-- Outline and primary Input -->
<div class="input list-outline list-primary">
  <input type="text" placeholder="Normal">
</div>
<!-- Filled and info Input -->
<div class="input list-filled list-info">
  <input type="text" placeholder="Normal">
</div>
You can use .list-{looks} to change the look input, available Looks {outline, filled}
