  • Type: class

  • Progressbar: .progressbar-{types}e.g: .progressbar.progressbar-primary, .progressbar.progressbar-error

Use .progressbar.progressbar-{types}
  • Progressbar Sizes
  • Type: class
  • Dependencies: .progressbar
  • Examples
  • Progressbar: .progressbar-{looks}e.g: .progressbar.progressbar-group, .progressbar.progressbar-dot
    <!-- Progressbar xs size and progressbar primary type-->
<div class="progressbar progressbar-primary progressbar-xs">
  <div class="progressbar-counting" data-counting="20%" style="width: 20%"></div>
</div>
<!-- Progressbar lg size and progressbar secondary type -->
<div class="progressbar progressbar-secondary progressbar-lg">
  <div class="progressbar-counting" data-counting="30%" style="width: 50%"></div>
</div>
You can use .progressbar-{sizes}
