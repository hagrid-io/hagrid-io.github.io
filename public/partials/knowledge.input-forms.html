The Inputs

Property DetailsVariationsDescription
  • Inputs Main Class
  • Type: class
  • Supports: [sizes, types, looks, displays]
  • Examples
  • Input: .inpute.g: .input
    <div class="input">
  <input type="text" placeholder="Normal">
</div>
Use .input class to reference a input element, the input class is a wappert to html inputs
  • Input types
  • Type: class
  • Dependencies: .input
  • Examples
  • Input: .input-{types}e.g: .input.btn-primary, .input.btn-error 
    <!-- primary input -->
<div class="input input-primary">
  <input type="text" placeholder="Primary Input">
</div>
<!-- error input -->
<div class="input input-error">
  <input type="text" placeholder="Error input">
</div>
Use .input.input-{types} classes to modify the apparence of a input
  • Input Sizes
  • Type: class
  • Dependencies: .input
  • Examples
  • Input: .input-{sizes}e.g: .input.input-sm, .input.input-lg
    <!-- a small input -->
<div class="input input-sm">
  <input type="text" placeholder="Small input">
</div>
<!-- a info type and large input -->
<div class="input input-info input-lg">
  <input type="text" placeholder="Info type and large input">
</div>
You can use .input.input-{sizes} to change size input, available Sizes {xs, sm, lg, xl}
  • Input Looks
  • Type: class
  • Dependencies: .input
  • Examples
  • Input: .input-{looks}e.g: .input.input-upper, .input.input-outline
    <!-- Outline and primary Input -->
<div class="input input-outline input-primary">
  <input type="text" placeholder="Normal">
</div>
<!-- Filled and info Input -->
<div class="input input-filled input-info">
  <input type="text" placeholder="Normal">
</div>
You can use .input-{looks} to change the look input, available Looks {outline, filled}

Main Class of Inputs

Code: 
<p>.input > input</p>
<div class="input">
  <input type="text" placeholder="Normal">
</div>
<p>.input.input-invalid > input.input-invalid</p>
<div class="input input-invalid">
  <input type="text" placeholder="Invalid" class="input-invalid">
</div>
<p>.input.input-disabled > input.input-disabled</p>
<div class="input input-disabled">
  <input type="text" placeholder="Disabled" class="input-disabled" disabled="disabled">
</div>
Result:

.input > input

.input.input-invalid > input.input-invalid

.input.input-disabled > input.input-disabled
