Commons and helper classes

When we building web project, normally we need some generic classes with specific properties, Hagrid has some that help us not specify in our css files

Table properties Helper Classes
Property DetailsVariationsDescription
  • Margins
  • Type: class
  • Supports: [axis, sizes, sides and measurements]
  • Examples
  • Sizes: m-[10-100]e.g: m-10, m-20, m-30...
    <div class="m-10">...</div>
  • Axis: m-{x, y}e.g: m-10-x, m-20-y
    <div class="m-10-x">...</div>
  • Sides: m-{top, right, bottom, left}e.g: m-10-top, m-20-right, m-30-bottom
    <div class="m-10-top">...</div>
  • Measurements: m-{px, percent, em, rem}e.g: m-percent, m-em, m-rem
    <div class="m-10 m-em">...</div>
  • Combined: sizes, axes, measurementse.g: class="m-10-x m-10-top m-percent"
    <div class="m-10-x m-10-top m-percent">...</div>
Hagrid supports margin classes to separate elements, .m-[sizes] to specify a size, .m-{measurements} to specify the measurement, by default measurements is px, .m-{axis} to specify a axes and .m-[side] to specify a side.
  • Paddings
  • Type: class
  • Supports: [axis, sizes, sides and measurements]
  • Examples
  • Sizes: p-[10-100]e.g: p-10, p-20, p-30...
    <div class="p-10">...</div>
  • Axis: p-{x, y}e.g: p-10-x, p-20-y
    <div class="p-10-x">...</div>
  • Sides: p-{top, right, bottom, left}e.g: p-10-top, p-20-right, p-30-bottom
    <div class="p-10-top">...</div>
  • Measurements: p-{px, percent, em, rem}e.g: p-percent, p-em, p-rem
    <div class="p-10 p-em">...</div>
  • Combined: sizes, axes, measurementse.g: class="p-10-x p-10-top p-percent"
    <div class="p-10-x p-10-top p-percent">...</div>
Hagrid supports paddings classes to add space inside of element, .p-[sizes] to specify a size, .p-{measurements} to specify the measurement, by default measurements is px, .p-{axis} to specify a axes and .p-[side] to specify a side.
  • Widths
  • Type: class
  • Supports: [sizes, devices]
  • Examples
  • Sizes: w-[10-100], percentagese.g: w-10, w-20, w-30...
    <div class="w-50">...</div>
  • Devices: w-[sizes]-{xs, sm, md, lg}e.g: w-10-xs, w-20-sm, w-30-md, w-40-lg
    <div class="w-100-xs w-50-lg">...</div>
You can use .w-[size] classes to specify a width of one web element, itÂ´s in percents, [10-100] , from 10 to 100, also you can specify a size to a device .w-[size]-{device} and transform this to multiples devices.
Note:  To layouts and columns you should use  grid system core
  • Colors
  • Type: class
  • Supports: [colors]
  • Examples
  • Color: {default, inverse, primary, secondary, info, warning, error}-colore.g: primary-color, secondary-color, inverse-color...
    <div class="primary-color">...</div>
You can use .{colors}-color class to specify one base color colors {default, inverse, primary, secondary, info, warning and error}
  • Backgrounds
  • Type: class
  • Supports: [background-colors]
  • Examples
  • Background Color: {default, inverse, primary, secondary, info, warning, error}-bgce.g: primary-bgc, secondary-bgc, inverse-bgc...
    <div class="primary-bgc">...</div>
You can use .{background-colors}-bgc class to specify one base background color background colors {default, inverse, primary, secondary, info, warning and error}
  • Looks
  • Type: class
  • Supports: [types-look]
  • Examples
  • Type Looks: {default, inverse, primary, secondary, info, warning, error}-looke.g: primary-look, secondary-look, inverse-look...
    <div class="primary-look">...</div>
You can use .{types}-look classes to specify one look to an element type looks {default, inverse, primary, secondary, info, warning and error}
  • Displays
  • Type: class
  • Supports: [displays]
  • example
  • Display: .d-{none, inline, inline-block, block, flex}e.g: d-none, d-block, d-inline-block
    <div class="d-none">...</div>
You can use .d-{displays} display classes to modify display element display {none, inline, inline-block, block, flex}
  • Aligns
  • Type: class
  • Supports: [aligns]
  • Examples
  • aligns: .align-{left, center, right, justify}e.g: align-left, align-center, align-right
    <div class="align-right">...</div>
You can use alings classes to aligns elements (inline, inline-block) with a block parentaligns {align-left, align-center, align-right, align-justfy}
  • Text Transform
  • Type: class
  • Supports: [text-transform]
  • Examples
  • aligns: .{uppercase, capitalize, lowercase, normal-text}e.g: align-left, align-center, align-right
    <h1 class="capitalize">...</h1>
You can use text transform classes to transform text Text Transforms {uppercase, lowercase, capitalize, normal-text}
