Grid System

Hagrid Grid System works mainly with percentages(layouts), Also the grid system supports flex box feature.

Table properties grid system
Property DetailsVariationsDescription
  • Grid Container
  • Type: class
  • Examples
  • Container: gse.g: gs or grid-system
    <div class="gs">...</div>
The main class to specify a grid system container
  • Layouts
  • Type: class
  • Supports: [sizes and devices]
  • Examples
  • layouts: layout-{sizes}e.g: layout-20, layout-40 
    <div class="gs">
  <div class="layout-60">...</div>
  <div class="layout-40">...</div>
</div>
  • Devices: layout-{devices}-{sizes}e.g: layout-sm-20, layout-lg-40 
    <div class="gs">
  <div class="layout-60 layout-sm-50">...</div>
  <div class="layout-40 layout-sm-50">...</div>
</div>
You must use .layout-{sizes} inside .gs container to distribute the area of the container to your need, also you can use .layout-{devices}-{sizes} class to modify layout size to specific devices, Device sizes, xs to small devices(smartphones), sm to medium screen(tablets), md to normal screens(laptop) and lg to large screens(Desktop or retina display).
  • Gutters
  • Type: class, attribute
  • Dependencies: .gs
  • Supports: [sizes and measurements]
  • Examples
  • Gutters Class: .gutterse.g: .gs.gutters
    <div class="gs gutters">
  <div class="layout-60">...</div>
  <div class="layout-40">...</div>
</div>
  • Gutters Attr: [gutters="{sizes}"]e.g: .gs[gutters="20px"], .gs[gutters="10%"] 
    <div class="gs" gutters="20px">
  <div class="layout-60">...</div>
  <div class="layout-40">...</div>
</div>
Gutters class and gutters attribute, affect directly to layout-{sizes} class children, gutters class for default is 1em of margin to each layout-{sizes} children, you can use gutter attribute to specify other sizes and measure, types of measure (px(10px, 20px and 30px), em(1em, 2em and 3em), and percent(10%, 20% and 30%))
  • Order
  • Type: class
  • Supports: [order]
  • Examples
  • Order: .ordere.g: .layout-50.order-2
    <div class="gs">
  <div class="layout-60 order-2">...</div>
  <div class="layout-40 order-1">...</div>
</div>
You can re order layouts without move html layouts
  • Horizontal Items Align
  • Type: class
  • Supports: [aligns]
  • Examples
  • Horizontal Aligns: justify-content-{horizontal-aligns}e.g: gs justify-content-flex-start
    <div class="gs justify-content-flex-start">
  <div class="layout-40">...</div>
  <div class="layout-40">...</div>
</div>
Hagrid supports horizonal aligns content, you can use .justify-content-{flex-start, flex-end, center, space-between, space-around}
  • Vertical Items Align
  • Type: class
  • Supports: [aligns]
  • Examples
  • Vertical Aligns: align-items-{horizontal-aligns}e.g: .gs.align-items-flex-start
    <div class="gs align-items-center">
  <div class="layout-40">...</div>
  <div class="layout-40">...</div>
</div>
  • Nested Layouts
  • Type: class
  • Examples
  • Nested: align-items-{horizontal-aligns}e.g: .gs.align-items-flex-start
    <div class="gs align-items-center">
  <div class="layout-40">...</div>
  <div class="layout-40">...</div>
</div>
Hagrid supports horizonal aligns content, you can use .justify-content-{flex-start, flex-end, center, baseline, stretch}
  • Grid Debug
  • Type: Attribute
  • Supports: [aligns]
  • Examples
  • Debug: [hagrid-debug]e.g: .gs.[hagrid-debug]
    <div class="gs" hagrid-debug>
  <div class="layout-30">...</div>
  <div class="layout-70">...</div>
</div>
You can use hagrid-debug to see layout dimentions

Grid system container

Code:
<!-- Short class .gs -->
<section class="gs">
  <!-- layouts statement-->
</section>
<!-- Short class .grid-system -->
<section class="grid-system">
  <!-- layouts statement-->
</section>

Grid system layouts

Code:
<!-- First Layout 30% and Second layout 70% -->
<section class="gs">
  <div class="layout-30"></div>
  <div class="layout-70"></div>
</section>
<!-- First Layout 20%, Second layout 50% and third layout 20% -->
<section class="gs">
  <div class="layout-20"></div>
  <div class="layout-50"></div>
  <div class="layout-30"></div>
</section>

Transform layouts for specific device size (width screen)
Code:
<section class="gs">
  <!-- For smartphones(xs) First layout width 100% and second layout width 100% -->
  <!-- For Tablets(sm), First layout width 50% and second layout width 50% -->
  <!-- For normal screens(md) first layout width 60% and second layout width 40% -->
  <!-- For large screens(lg) first layout width 80% and second layout width 20% -->
  <div class="layout-xs-100 layout-sm-50 layout-md-60 layout-lg-80"></div>
  <div class="layout-xs-100 layout-sm-50 layout-md-40 layout-lg-20"></div>
</section>
