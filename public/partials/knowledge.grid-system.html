Grid System

Hagrid Grid System works mainly with percentages(layouts), Also the grid system supports flex box feature.

Table properties grid layout
PropertyTypeDescriptionExamples
gs, grid-systemclassClass container to grid systemexample
layout-classThe main class to specify a layout, layout-40example
-xs-, -sm, -md-, -lg-classDevice sizes, xs to small devices(smartphones), sm to medium screen(tablets), md to normal screens(laptop) and lg to large screens(Desktop or retina display), eg layout-xs-40, layout-lg-60example
guttersclassclass to separate layoutsexample
gutters=[1em]attrYou can use tree option to separate layouts, it is in tree types of measure (px, em, and percent)example
order-classYou can re order layouts without move html layoutsexample
gs-items-alignsclassHorizontal Alignsexample
gs-items-aligns.gs-vertical-orientationclassVertical Alignsexample
hagrid-debugattrYou can use hagrid-debug to see layout dimentionsexample

Grid system container and layouts

Code:
<!-- Short class .gs -->
<section class="gs">
  <!-- layouts statement-->
</section>
<!-- Short class .grid-system -->
<section class="grid-system">
  <!-- layouts statement-->
</section>
Code:
<!-- First Layout 30% and Second layout 70% -->
<section class="gs">
  <div class="layout-30"></div>
  <div class="layout-70"></div>
</section>
<!-- First Layout 20%, Second layout 50% and third layout 20% -->
<section class="gs">
  <div class="layout-20"></div>
  <div class="layout-50"></div>
  <div class="layout-30"></div>
</section>
Result:

Sizes and Transform layouts

Code:
<section class="gs">
  <div class="layout-lg-80"></div>
  <div class="layout-lg-20"></div>
</section>
Result:
Transform layouts for specific device size (width screen)
Code:
<section class="gs">
  <!-- For smartphones(xs) First layout width 100% and second layout width 100% -->
  <!-- For Tablets(sm), First layout width 50% and second layout width 50% -->
  <!-- For normal screens(md) first layout width 60% and second layout width 40% -->
  <!-- For large screens(lg) first layout width 80% and second layout width 20% -->
  <div class="layout-xs-100 layout-sm-50 layout-md-60 layout-lg-80"></div>
  <div class="layout-xs-100 layout-sm-50 layout-md-40 layout-lg-20"></div>
</section>
Result:

Order columns

Code:
<section class="gs">
  <div class="layout-50 order-2"></div>
  <div class="layout-20 order-3"></div>
  <div class="layout-30 order-1"></div>

Gutters

gutters class adds a margin of 1em to each layout column

Code:
<section class="gs gutters">
  <div class="layout-10"></div>
  <div class="layout-20"></div>
  <div class="layout-70"></div>
</section>
