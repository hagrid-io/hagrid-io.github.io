Images and Avatars

Hagrid has some class to normalize look images and avatars.

Table properties Images
Property DetailsVariationsDescription
  • Images
  • Type: class
  • Main Class: .img
  • Supports: [styles]
  • Examples
  • Styles: .img-{circle, radius}e.g: .img-circle, .img-radius
    <figure class="img img-circle">
  <img src="" alt="Image"/>
</figure>
Hagrid supports some images classes to change style images .img-{circle, radius}

Image Styles

Code:
<!-- Image Circle-->
<!--- divs is not required -->
<div>
  <figure class="img img-circle">
    <img src="http://dummyimage.com/200x200" alt="Image"/>
  </figure>
</div>
<!-- Image Radius-->
<div class="div-is-not-required">
  <figure class="img img-radius">
    <img src="http://dummyimage.com/200x200" alt="Image"/>
  </figure>
</div>
