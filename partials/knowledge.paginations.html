Paginations

The Paginations

Property DetailsVariationsDescription
  • Pagination Main Class
  • Type: class
  • Supports: [types, looks, sizes]
  • Examples
  • Pagination: .paginatione.g: .pagination
    <nav class="pagination">
  <a href="">1</a>
  <a href="">2</a>
  <a href="">3</a>
  <a href="">4</a>
</nav>
Use .list class to reference a list element.
  • Pagination types
  • Type: class
  • Dependencies: .pagination
  • Examples
  • Pagination: .pagination-{types}e.g: .pagination.pagination-primary, .pagination.pagination-error 
    <!-- pagination pagination-primary -->
<nav class="pagination pagination-primary">
  <span class="pagination-prev disable"></span>
  <a href="">1</a>
  <a href="" class="current">2</a>
  <a href="">3</a>
  <a href="">4</a>
  <span class="pagination-next"></span>
</nav>
<!-- pagination pagination-warning -->
<nav class="pagination pagination-warning">
  <span class="pagination-prev disable"></span>
  <a href="">1</a>
  <a href="" class="current">2</a>
  <a href="">3</a>
  <a href="">4</a>
  <span class="pagination-next"></span>
</nav>
Use .list.list-{types} classes to modify the apparence of a list
  • List Looks
  • Type: class
  • Dependencies: .list
  • Examples
  • Pagination: .list-{looks}e.g: .list.list-group, .list.list-dot
    <!-- List group -->
<ul class="list list-group">
  <li>Lorem ipsum dolor sit amet</li>
  <li>Lorem ipsum dolor sit amet</li>
  <li>Lorem ipsum dolor sit amet</li>
  <li>Lorem ipsum dolor sit amet</li>
</ul>
<!-- List dot and primary -->
<ul class="list list-dot list-primary">
  <li>Lorem ipsum dolor sit amet</li>
  <li>Lorem ipsum dolor sit amet</li>
  <li>Lorem ipsum dolor sit amet</li>
  <li>Lorem ipsum dolor sit amet</li>
</ul>
You can use .list-{looks} to change the look of list, available Looks {dot, circle, square, cube, arrow, check, group, divider, border-left}
