Dropdowns

Dropdowns works how html selector, you can use dropdowns when you needs user nested list inside a button.

Table properties Dropdowns
Property DetailsVariationsDescription
  • Dropdown
  • Type: class
  • Supports: [types, looks, events]
  • Examples
  • Button: .btne.g: .btn
    <div class="dropdown" hagrid-component="dropdown">
  <a href="#" class="btn btn-primary" hagrid-role="dropdown">Dropdown A<span class="icon-dropdown icon-keyboard_arrow_down"></span></a>
  <ul class="dropdown-menu">
    <li><a href="#">Dropdown Menu 1</a></li>
    <li><a href="#">Dropdown Menu 2</a></li>
    <li><a href="#">Dropdown Menu 3</a></li>
    <li><a href="#">Dropdown Menu 4</a></li>
    <li><a href="#">Dropdown Menu 5</a></li>
  </ul>
</div>
